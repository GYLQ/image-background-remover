常见问题

快速解答您的疑问

每天有多少次免费使用？
未登录用户每天可免费使用3次。登录后每天仍有3次免费额度，另外注册即送3积分（永久）。付费用户根据套餐每月获得100-2000积分。
积分会过期吗？
积分永不过期。未使用的积分会累积到下个月，无论您是免费用户还是付费用户。例如您有15积分，用了5次，剩余10积分仍可继续使用。
付费方案如何购买？
我们即将支持 PayPal 和信用卡购买。在那之前，您可以通过积分包（一次性购买）来补充积分，享受更多使用额度。
去背景后的图片有水印吗？
免费用户输出的图片会带有小水印。升级到付费方案（基础版及以上）即可去除水印，获得纯净的高清图片。
支持哪些图片格式？
支持 JPEG、PNG 和 WebP 格式。最大支持 5MB 的图片文件。
如何升级或取消订阅？
登录后进入个人中心，可以查看当前套餐和用量。如需升级或变更方案，请联系我们。取消订阅后您仍可使用剩余积分，不会被清零。
个人隐私是否安全？
我们非常重视隐私。上传的图片会在处理完成后自动删除，不会保存到任何服务器。我们也不会存储您的 Google 账号信息（仅用于登录验证）。
可以使用 API 吗？
API 访问即将向专业版和团队版用户开放。通过 API 您可以将去背景功能集成到自己的工作流程中，实现自动化处理。
