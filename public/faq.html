常见问题

快速解答您的疑问

注册用户有多少免费额度？
注册即送3积分（一次性），用于去背景处理。无需每日签到或消耗，无有效期限制。
积分会过期吗？
积分永不过期。购买积分包后，积分将一直保存在账户中，直到用完为止。
如何购买积分？
我们支持 PayPal 和信用卡购买。一次性购买积分包，积分永不过期。用完再买，方便灵活。
支持哪些图片格式？
支持 JPEG、PNG 和 WebP 格式。最大支持 5MB 的图片文件。
如何查看我的积分余额？
登录后点击首页右上角或进入个人中心，即可查看当前积分余额和使用记录。
用完积分怎么办？
积分用完后，您可以购买积分包补充。我们支持 PayPal 和信用卡，支付成功后积分立即到账。
个人隐私是否安全？
我们非常重视隐私。上传的图片会在处理完成后自动删除，不会保存到任何服务器。我们也不会存储您的 Google 账号信息（仅用于登录验证）。
可以集成 API 吗？
API 访问即将开放。通过 API 您可以将去背景功能集成到自己的工作流程中，实现自动化处理。
