常见问题

积分包和月费会员有什么区别？
积分包是一次性购买，积分永不过期，适合不规律使用。月费会员每月自动获得固定积分，当月有效，适合经常使用的用户。月费用户也可以同时购买积分包，积分包永不过期。
积分会过期吗？
购买所得积分永不过期。注册赠送的 3 积分也是永久有效。月费会员每月到账的积分在当月有效，月底清零，不影响已购买的积分包。
如何购买？
点击「立即购买」后会跳转支付页面。我们即将支持 PayPal 和信用卡支付。在此之前，如需购买可以联系我们手动处理。
可以退款吗？
积分购买后如未使用，可联系客服申请退款。已消耗的积分无法退款。月费会员如需取消，可在下次扣费前联系我们。
每次使用消耗多少积分？
每处理一张图片，无论尺寸大小，消耗 1 积分。月费用户在当月可获得固定积分，积分包用户可永久累积使用。
API 访问是什么？
API 访问即将向专业版和团队版用户开放。通过 API 您可以将去背景功能集成到自己的网站、工作流程或应用中，实现自动化批量处理。
